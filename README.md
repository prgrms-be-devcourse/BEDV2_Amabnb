## 🔹 기술 스택
* java 17  
* Spring Boot 2.7.0  
* Gradle 7.4.1  
* data Jpa
* QueryDSL  
* Amazon Aurora MySQL
* Rest-Docs  
* Junit 5  
* Jwt  
* Oauth2  
* SpringSecurity  
* AWS S3  
* Jenkins  

<br/>

## 🔹 기능 정의
- SNS 회원가입
- 서비스 가입자 대상 서비스 이용 정책 (비회원 서비스 접근 제한)
- 토큰을 사용한 서비스 이용 관리
- 로그인/로그아웃
- 회원탈퇴
- 숙소 등록, 수정, 삭제, 검색
- 예약 생성, 수정, 삭제, 조회
- 리뷰 작성 및 수정, 삭제, 조회
- 이미지 업로드 기능
- 지속적인 통합(Continuous Integration)구축
